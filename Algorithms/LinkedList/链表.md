# 链表 
链表是一种线性数据结构，其中元素以节点的形式存储，每个节点包含数据和指向下一个节点的指针。链表可以是单向的（单链表）或双向的（双链表），也可以是循环的（循环链表）。

链表的入口节点被称为链表的头节点也就是head。
## 链表的类型 
### 单链表 
单链表是一种线性数据结构，其中元素以节点的形式存储，每个节点包含数据和指向下一个节点的指针。单链表只能从头节点开始遍历，无法从尾节点开始遍历。
### 双链表 
每个节点有两个指针域，分别指向下一个节点和上一个节点。双链表可以实现双向遍历，即可以从头节点开始遍历，也可以从尾节点开始遍历。![[双链表.png]]
### 循环链表 
循环链表是一种特殊的链表，其中最后一个节点的指针域指向头节点，形成一个循环结构。循环链表可以实现双向遍历，即可以从头节点开始遍历，也可以从尾节点开始遍历。![[循环链表.png]]
## 链表的存储方式
数组在内存中是连续分布的，链表在内存中是不连续分布的。链表是通过指针域的指针链接在内存中的各个节点。 链表的存储方式可以分为静态链表和动态链表。静态链表是在编译时就分配好内存空间的链表，动态链表是在运行时动态分配内存空间的链表。

## 链表的操作 
### 删除节点 O(1)
将要删除的节点的前一个节点的指针指向后一个节点，被删除的节点在Java、Python中会自动回收![[链表删除节点.png]]
### 插入节点 O(1)
在链表的头部插入节点，将新节点的指针域指向头节点，头节点指向新节点。在链表内部插入节点，将上一个节点的指针域指向新节点，新节点的指针域指向下一个节点。在链表的尾部插入节点，将新节点的指针域指向None，将尾节点的指针域指向新节点。![[链表添加节点.png]]
## 与数组的对比
数组长度固定，如果需要改动数组长度，需要定义一个新的数组。
链表的长度可以是不固定的，并且可以动态增删，适合数据量不固定，频繁增删，较少查询的场景。
## 移除链表元素 

### 递归思路
时间复杂度O(n) 空间复杂度O(n)
```Python 
# 链表节点定义 
class Node: 
	def __init__(self, val, next=None): 
		self.val = val 
		self.next = next
		
def removeElements(head: Node, val): 
	# 基础情况，空链表，不需要删除
	if not head: 
		return None 
	# 递归情况：先处理后面的链表
	# 头节点要删，答案就是后续节点递归的结果 
	elif head.val == val: 
		return removeElements(head.next, val) 
	# 头节点不删，把头节点和后续节点拼接起来 
	else: 
		head.next = removeElements(head.next, val) 
		return head 
```
### 删除原链表 
时间复杂度O(n) 空间复杂度O(1) 
```Python 
# 链表节点定义 
class Node: 
	def __init__(self, val, next=None): 
		self.val = val 
		self.next = next
		
def removeElements(head: Node, val): 
	# 处理头节点为目标节点的情况，一直删到头不是val 
	while head and head.val == val: 
		head = head.next # 直接移动头指针，删掉原来的头 
	# 现在 head 要么为空，要么值 != val 
	cur = head 
	# 开始遍历，删除中间/尾部节点
	while cur and cur.next: 
		if cur.next.val == val: 
			cur.next = cur.next.next # 直接跳过 next 节点：原地删除
		else: 
			cur = cur.next 
	return head 
```
### 虚拟头结点(推荐)
时间复杂度O(n) 空间复杂度O(1) 
```Python 
# 链表节点定义 
class Node: 
	def __init__(self, val, next=None): 
		self.val = val 
		self.next = next 

def removeElements(head: Node, val): 
	# 创建虚拟头结点，统一处理所有节点
	dummy_node = Node(0) 
	dummy_node.next = head 
	cur = dummy_node # 从虚拟头开始遍历
	while cur and cur.next: 
		if cur.next.val == val: 
			cur.next = cur.next.next # 统一直接跳过 next 节点：原地删除 
		else: 
			cur = cur.next 
	# 最终返回真实头结点
	return dummy_node.next 
```
## 设计链表 
在链表类中实现这些功能：

- get(index)：获取链表中第 index 个节点的值。如果索引无效，则返回-1。
- addAtHead(val)：在链表的第一个元素之前添加一个值为 val 的节点。插入后，新节点将成为链表的第一个节点。
- addAtTail(val)：将值为 val 的节点追加到链表的最后一个元素。
- addAtIndex(index,val)：在链表中的第 index 个节点之前添加值为 val  的节点。如果 index 等于链表的长度，则该节点将附加到链表的末尾。如果 index 大于链表长度，则不会插入节点。如果index小于0，则在头部插入节点。
- deleteAtIndex(index)：如果索引 index 有效，则删除链表中的第 index 个节点。
### 单向链表 
```Python 
class Node:
	def __init__(self, val, next=None):
		self.val = val
		self.next = next 
	
class LinkedList:
	def __init__(self):
		self.head = None 
		self.size = 0 
	
	def get(self, index): 
	    if index < 0 or index >= self.size: 
					return -1 
			cur = self.head 
			for _ in range(index): 
			    cur = cur.next 
			return cur.val 
	
	def addAtHead(self, val): 
	    new_node = Node(val, self.head) 
	    self.head = new_node 
	    self.size += 1 
	
	def addAtTail(self, val): 
	    new_node = Node(val) 
	    if self.head is None: 
	        self.head = new_node 
	    else: 
	        cur = self.head 
	        while cur.next: 
	            cur = cur.next 
	        cur.next = new_node 
	    self.size += 1 
	
	def addAtIndex(self, index, val): 
	    if index < 0 or index > self.size: 
	        return 
	    if index == 0: 
	        self.addAtHead(val) 
	    else: 
	        new_node = Node(val) 
	        cur = self.head 
	        for _ in range(index - 1): 
	            cur = cur.next 
	        new_node.next = cur.next 
	        cur.next = new_node 
	        self.size += 1 
	
	def deleteAtIndex(self, index): 
	    if index < 0 or index >= self.size: 
	        return 
	    if index == 0: 
	        self.head = self.head.next 
	    else: 
	        cur = self.head 
	        for _ in range(index - 1): 
	            cur = cur.next 
	        cur.next = cur.next.next 
	    self.size -= 1
```

## 反转链表 
### 迭代 
```python 
# 时间复杂度O（n）空间复杂度O(1)
def reverse_list(head):
    prev = None
    cur = head
    while cur:
        # 保存下一个节点
        temp = cur.next
        # 反转当前节点的指针
        cur.next = prev
        # 更新prev和cur
        prev = cur
        cur = temp
    return prev
```
### 递归 
```Python 
# 时间复杂度O（n）空间复杂度O(n)
def reverse_list(head):
    # 空链表，直接返回；到达链表末尾（最后一个节点），直接返回
    if not head or not head.next:
        return head
    new_head = reverse_list(head.next)
    # 让下一个节点指回当前节点
    head.next.next = head
    # 断开当前节点向前的连接
    head.next = None
    # 始终返回最后一个节点
    return new_head
```
## 交换节点
[力扣题目链接](https://leetcode.cn/problems/swap-nodes-in-pairs/description/)
给定一个链表，两两交换其中相邻的节点，并返回交换后的链表。

你不能只是单纯的改变节点内部的值，而是需要实际的进行节点交换。 
### 迭代 
```Python 
# 时间复杂度O（n）空间复杂度O(1)
def swap_pairs(head):
    dummy_head = ListNode(0, next=head)
    cur = dummy_head
    while cur.next and cur.next.next:
        # 保存下一个节点和下下下个节点
        temp0 = cur.next
        temp1 = cur.next.next.next
        # 进行交换
        cur.next = cur.next.next
        cur.next.next = temp0
        temp0.next = temp1
        cur = temp0.next
    return dummy_head.next
``` 
### 递归 
```Python 
# 时间复杂度O（n）空间复杂度O(n)
def swap_pairs(head):
    # 1. 基本情况（终止条件）
    # 空链表，直接返回；到达链表末尾（最后一个节点），直接返回
    if not head or not head.next:
        return head
    
    # 2. 递归调用：处理后面的子链表
    new_head = swap_pairs(head.next.next)  # 递归处理从第3个节点开始的子链表
    
    # 3. 执行交换操作
    # 让下一个节点指回当前节点
    head.next.next = head
    # 断开当前节点向前的连接
    head.next = new_head
    
    # 4. 返回新的头节点
    return head.next

```
## 删除链表的倒数第N个节点 
[力扣题目链接](https://leetcode.cn/problems/remove-nth-node-from-end-of-list/description/)
给定一个链表，删除链表的倒数第 n 个节点，并且返回链表的头结点。

进阶：你能尝试使用一趟扫描实现吗？ 
### 双指针 
```Python 
# 时间复杂度O（n）空间复杂度O(1)
def remove_nth_from_end(head, n):
    dummy_head = ListNode(0, next=head)
    slow = fast = dummy_head
    # 快指针先走n+1步
    for _ in range(n + 1):
        fast = fast.next
    # 快慢指针同时移动，直到快指针到达链表末尾
    while fast:
        slow = slow.next
        fast = fast.next
    # 此时慢指针的下一个节点就是要删除的节点
    slow.next = slow.next.next
    return dummy_head.next
```
## 链表相交 
[力扣题目链接](https://leetcode.cn/problems/intersection-of-two-linked-lists-lcci/)
给你两个单链表的头节点 headA 和 headB ，请你找出并返回两个单链表相交的起始节点。如果两个链表没有交点，返回 null 。
### 双指针法
```Python 
def get_intersection_of_two(headA, headB): 
    # 走完两条链表,通过切换链表来平衡路径长度差
    pA, pB = headA, headB
    while pA != pB:
        pA = pA.next if pA else headB
        pB = pB.next if pB else headA
    return pA    
```
### 先算长度差
```Python
def get_intersection_of_two(headA, headB):
    # 计算长度
    lenA, lenB = 0, 0
    pA, pB = headA, headB
    while pA:
        lenA += 1
        pA = pA.next
    while pB:
        lenB += 1
        pB = pB.next
    # 长链表先走差值步
    diff = abs(lenA - lenB)
    if lenA > lenB:
        for _ in range(diff):
            headA = headA.next
    else:
        for _ in range(diff):
            headB = headB.next
    # 一起走直到相遇
    while headA != headB:
        headA = headA.next
        headB = headB.next
    return headA
```
## 环形链表 
[力扣题目链接](https://leetcode.cn/problems/linked-list-cycle-ii/)
给定一个链表，返回链表开始入环的第一个节点。 如果链表无环，则返回 null。 
为了表示给定链表中的环，使用整数 pos 来表示链表尾连接到链表中的位置（索引从 0 开始）。 如果 pos 是 -1，则在该链表中没有环。![[环形链表.png]]
# 哈希集合
```Python 
def detect_cycle(head): 
    visited = set() 
    cur = head 
    while cur: 
        if cur in visited: 
            return cur 
        visited.add(cur) 
        cur = cur.next 
    return None  
``` 
# 快慢指针 
```Python 
def detect_cycle(head): 
    slow = fast = head 
    while fast and fast.next: 
        slow = slow.next 
        fast = fast.next.next 
        if slow = fast: 
            # 有环，找出口 
            p = head 
            while p != slow: 
                p = p.next 
                slow = slow.next 
            return p 
    return None 
```
