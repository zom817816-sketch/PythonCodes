## 理论基础 
- 定义：**数组是存放在连续内存空间上的相同类型数据的集合**（Python中的array.array和numpy.array）
	- **数组下标都是从0开始的** 
	- **数组内存空间的地址是连续的**
正是**因为数组在内存空间的地址是连续的，所以我们在删除或者增添元素的时候，就难免要移动其他元素的地址。**
## 二分查找 

**题目**：给定一个 n 个元素有序的（升序）整型数组 nums 和一个目标值 target  ，写一个函数搜索 nums 中的 target，如果目标值存在返回下标，否则返回 -1。
**这道题目的前提是数组为有序数组**，同时题目还强调**数组中无重复元素**，因为一旦有重复元素，使用二分查找法返回的元素下标可能不是唯一的 
### 二分法第一种写法\[left, right] 
- while (left <= right) 要使用 <= ，因为left == right是有意义的，所以使用 <= 
- if (nums\[middle] > target) right 要赋值为 middle - 1，因为当前这个nums\[middle]一定不是target，那么接下来要查找的左区间结束下标位置就是 middle - 1![[二分查找_左闭右闭.jpg]]
```python 
def binary_search(nums, target): 
	left = 0 
	right = len(nums) - 1 
	while left <= right: 
		middle = (left + right) // 2 
		if nums[middle] == target: 
			return middle # 找到目标，返回下标
		elif nums[middle] < target: 
			left = middle + 1 # 目标在右区间，更新左边界
		else: 
			right = middle - 1 # 目标在左区间，更新右边界
	return -1 # 未找到目标，返回-1
```
### 二分法第二种写法\[left, right) 
- while (left < right)，这里使用 < ,因为left == right在区间\[left, right)是没有意义的 
  - if (nums\[middle] > target) right 更新为 middle，因为当前nums\[middle]不等于target，去左区间继续寻找，而寻找区间是左闭右开区间，所以right更新为middle，即：下一个查询区间不会去比较nums\[middle]![[二分查找_左闭右开.jpg]]
```python
def binary_search(nums, target): 
	left = 0 
	right = len(nums) - 1 
	while left < right: 
		middle = (left + right) // 2 
		if nums[middle] == target: 
			return middle # 找到目标，返回下标
		elif nums[middle] < target: 
			left = middle + 1 # 目标在右区间，更新左边界
		else: 
			right = middle # 目标在左区间，更新右边界
	return -1 # 未找到目标，返回-1
```
区间的定义就是不变量，那么在循环中坚持根据查找区间的定义来做边界处理，就是循环不变量规则。
## 移除元素 
[Leetcode链接](https://leetcode.cn/problems/remove-element/)
给你一个数组 nums 和一个值 val，你需要 原地 移除所有数值等于 val 的元素，并返回移除后数组的新长度。

不要使用额外的数组空间，你必须仅使用 O(1) 额外空间并**原地**修改输入数组。

**要知道数组的元素在内存地址中是连续的，不能单独删除数组中的某个元素，只能覆盖。**
### 暴力解法 
两层for循环，一个for循环遍历数组元素 ，第二个for循环更新数组 
```Python 
	def remove(nums, target): 
		length = len(nums) 
		k = 0 
		for i in range(length): 
			if nums[i] == target: 
				k += 1 
				for j in range(i, length-1): 
					nums[j] = nums[j+1] 
		return k 	
```
### 双指针法
双指针法（快慢指针法）：**通过一个快指针和慢指针在一个for循环下完成两个for循环的工作。**
定义快慢指针 
- 快指针：寻找新数组的元素 ，新数组就是不含有目标元素的数组
- 慢指针：指向更新 新数组下标的位置
**双指针法（快慢指针法）在数组和链表的操作中是非常常见的，很多考察数组、链表、字符串等操作的面试题，都使用双指针法。** 
```Python 
	def remove(nums, target): 
		# 快慢指针 
		length = len(nums) 
		fast, slow = 0, 0 
		while fast < length: 
		    if nums[fast] != target: 
						nums[slow] = nums[fast] 
						slow += 1 
				fast += 1 
		return slow 
```
```Python 
	def remove(nums, target): 
		# 双向指针 
		length = len(nums) 
		left, right = 0, length-1 
		# 左右指针未相遇时持续遍历
		while left <= right: 
			# 左指针右移：跳过所有「不需要删除」的元素，直到找到待删元素或指针越界
			while left <= right and nums[left] != target: 
				left += 1 
			# 右指针左移：跳过所有「需要删除」的元素，直到找到保留元素或指针越界
			while left <= right and nums[right] == target: 
				right -= 1 
			# 若左右指针未相遇，用右指针的保留元素覆盖左指针的待删元素
			while left < right: 
				nums[left] = nums[right]
		return left # 左指针最终位置 = 新数组有效长度 			
```
## 有序数组的平方 
[Leetcode链接](https://leetcode.cn/problems/squares-of-a-sorted-array/)
给你一个按 **非递减顺序** 排序的整数数组 nums，返回 每个数字的平方 组成的新数组，要求也按 **非递减顺序** 排序。
### 暴力解法 
先对数组中的每个元素求平方，再对数组排序，时间复杂度为O(nlogn+n) = O(nlogn) 
```Python 
	def sortedSquares(nums): 
		new_nums = nums.copy()
		for i in range(len(nums)): 
			new_nums[i] = nums[i]**2 
		new_nums.sort() 
		return new_nums 
	# 列表推导式 
	# return sorted([num**2 for num in nums])
```
### 双指针法 
数组平方的最大值一定在数组的两端，不是最左边就是最右边，不可能是中间。
定义一个新数组result，和A数组一样的大小，让k指向result数组终止位置。
如果A[i] * A[i] < A[j] * A[j] 那么result[k--] = A[j] * A[j]
如果A[i] * A[i] >= A[j] * A[j] 那么result[k--] = A[i] * A[i]
```Python 
	def sortedSquares(nums): 
		length = len(nums) 
		left, right = 0, length-1 
		result = [0] * length 
		k = length-1 
		while left <= right: 
			if nums[left]**2 > nums[right]**2: 
				result[k] = nums[left]**2 
				left += 1 
			else: 
				result[k] = nums[right]**2 
				right -= 1 
			k -= 1 
		return result 
```
## 长度最小的子数组 
[力扣题目链接](https://leetcode.cn/problems/minimum-size-subarray-sum/) 
给定一个含有 n 个正整数的数组和一个正整数 s ，找出该数组中满足其和 ≥ s 的长度最小的连续子数组，并返回其长度。如果不存在符合条件的子数组，返回 0。
### 暴力解法 
使用两层for循环，遍历列表中所有的子数组情况，并找出满足要求的最小子数组的长度 
```Python 
def minSubArrayLen(nums, target): 
	if sum(nums) < target: 
		return 0 
	length = len(nums) 
	min_len = length + 1 
	for i in range(0, length): 
		sum_ = 0 
		for j in range(i, length): 
			sum_ += nums[j] 
				if sum_	>= target: 
					sub_len = j - i + 1 
					if sub_len < min_len: 
						min_len = sub_len 
					break 
	return min_len if min_len != length + 1 else 0 
```
### 滑动窗口（双指针） 
所谓滑动窗口，就是不断的调节子序列的起始位置和终止位置，从而得出我们想要的结果。 
```Python 
def minSubArrayLen(nums, target): 
	length = len(nums) 
	left, right = 0, 0 
	min_len = length + 1 
	cur_sum = 0 # 当前的累加值 

	while right < length:
		cur_sum += nums[right] 

		while cur_sum >= target: 
			min_len = min(min_len, right-left+1) 
			cur_sum -= nums[left] 
			left += 1 

		right += 1 

	return 0 if min_len == length + 1 else min_len
```
## 螺旋矩阵II 
[Leetcode链接](https://leetcode.cn/problems/spiral-matrix-ii/)
给定一个正整数 n，生成一个包含 1 到 n^2 所有元素，且元素按顺时针顺序螺旋排列的正方形矩阵。
示例:
输入: 3 输出: [ [ 1, 2, 3 ], [ 8, 9, 4 ], [ 7, 6, 5 ] ] 
![[螺旋矩阵2.png]]
这里一圈下来，我们要画每四条边，这四条边怎么画，每画一条边都要**坚持一致的左闭右开**，或者左开右闭的原则，这样这一圈才能按照统一的规则画下来。
```Python 
def generate_matrix(n): 
	nums = [[0] * n for _ in range(n)] 
	start_x, start_y = 0, 0 # 起始点
	loop, mid = n // 2, n // 2 # 迭代次数 n为奇数时的中心点
	count = 1 # 计数 
	
	for offset in range(1, loop+1): # 每循环一层偏移量加1，偏移量从1开始
		for i in range(start_y, n-offset):  # 从左至右，左闭右开
			nums[start_x][i] = count 
			count += 1 
		for i in range(start_x, n-offset): # 从上至下
			nums[i][n-offset] = count 
			count += 1 
		for i in range(n-offset, start_y, -1): # 从右至左
			nums[n-offset][i] = count 
			count += 1
		for i in range(n-offset, start_x, -1): # 从下至上
			nums[i][start_y] = count 
			count += 1
		start_x += 1 # 更新起始点
		start_y += 1 	
	
	if n % 2 != 0: 
		nums[mid][mid] = count # n为奇数时，填充中心点
	
	return nums 
```
第二写法设置四个边界，每次需要走完所有边界 
```Python 
def generate_matrix(n): 
	if n <= 0: 
		return [] 
		
	# 初始化矩阵 
	matrix = [[0] * n for _ in range(n)] 
	
	# 初始化边界和起始值 
	top, bottom, left, right = 0, n-1, 0, n-1 
	num = 1 
	
	while top <= bottom and left <= right: 
		# 从左到右填充上边界 
		for i in range(left, right+1): 
			matrix[top][i] = num 
			num += 1 
		top += 1 
		# 从上到下填充右边界 
		for i in range(top, bottom+1): 
			matrix[right][i] = num 
			num += 1
		right -= 1 
		# 从右到左填充下边界 
		for i in range(right, left-1, -1): 
			matrix[bottom][i] = num 
			num += 1 
		bottom -= 1 
		# 从下到上填充左边界 
		for i in range(bottom, top-1, -1): 
			matrix[left][i] = num 
			num += 1 
		left -= 1 
	
	return matrix	 		
```
## 区间和 
题目描述：
给定一个整数数组 Array，请计算该数组在每个指定区间内元素的总和。
输入描述：
第一行输入为整数数组 Array 的长度 n，接下来 n 行，每行一个整数，表示数组的元素。随后的输入为需要计算总和的区间，直至文件结束。
输出描述：
输出每个指定区间内元素的总和。
### 暴力解法 
在给定区间下，对区间内的元素进行累加
在极端情况下，即查询m次，每次区间范围为0到n-1，导致算法时间复杂度为O(m * n) 
```Python 
def main(): 
	n = int(input()) 
	array, result = [], []
	for _ in range(n): 
		array.append(int(input())) 
	while True: 
		try:
			interval = list(map(int,input().split())) 
			a, b = interval[0], interval[1]
			interval_sum = sum(array[a: b+1])
			result.append(interval_sum) 
		except EOFError: 
			break
	return result 
for res in main(): 
	print(res)
```

### 前缀和 
前缀和的思想是重复利用计算过的子数组之和，从而降低区间查询需要累加计算的次数。**在涉及计算区间和的问题时非常有用**。
在数组上下标i到j之间的累加和，可以用sum[0:j] - sum[0:i]代替
```Python 
def main(): 
    # 一次性读取所有输入，切成字符串列表（不管换行，全部当成一串数字）
    import sys 
    input = sys.stdin.read().split() 
    ptr = 0 # 指针：记录当前读到的位置 
    
    # 读n个数组元素 
    arr = [] 
    for _ in range(n): 
        arr.append(int(input[ptr]))
        ptr += 1
        
    # 构建前缀和数组（核心：快速求区间和）
    # pre[0] = 0 
    # pre[i] 表示前i个数之和：arr[0] + arr[1] + ... + arr[i-1]
    pre = [0] * (n + 1)
    for i in range(n): 
        pre[i+1] = pre[i] + arr[i]
        
    # 处理所有区间查询，直到读完所有数据
    while ptr < len(input): 
        a, b = int(input[ptr]), int(input[ptr+1]) 
        ptr += 2
        
        # 求 [a, b] 闭区间的和（0 下标）
        # 公式：sum(a~b) = pre[b+1] - pre[a]
        print(pre[b+1] - pre[a])
```
## 开发商购买土地 
题目描述：
在一个城市区域内，被划分成了n * m个连续的区块，每个区块都拥有不同的权值，代表着其土地价值。目前，有两家开发公司，A 公司和 B 公司，希望购买这个城市区域的土地。现在，需要将这个城市区域的所有区块分配给 A 公司和 B 公司。然而，由于城市规划的限制，只允许将区域按横向或纵向划分成两个子区域，而且每个子区域都必须包含一个或多个区块。为了确保公平竞争，你需要找到一种分配方式，使得 A 公司和 B 公司各自的子区域内的土地总价值之差最小。
> 注意：区块不可再分。

输入描述: 
第一行输入两个正整数，代表 n 和 m。接下来的 n 行，每行输出 m 个正整数。
输出描述: 
请输出一个整数，代表两个子区域内土地总价值之间的最小差距。
```Python
def main(): 
    import sys
    input = sys.stdin.read().split() 
    idx = 0 
    
    # 读取矩阵行数和列数 
    n, m = int(input(idx)), int(input(idx + 1)) 
    idx += 2 
    
    matrix = [] # 存储原矩阵 
    total = 0 # 矩阵之和 
    
    # 读取矩阵数据，并计算矩阵和 
    for i in range(n): 
        row = [] 
        for j in range(m): 
            num = int(input[idx]) 
            idx += 1 
            row.append(num) 
            total += num 
        matrix.append(row) 
    
    # 计算每一行的和（横向）
    horizontal = [0] * n 
    for i in range(n): 
        for j in range(m): 
            horizontal[i] += matrix[i][j]
    # 计算每一列的和（纵向） 
    vertical = [0] * m 
    for j in range(m): 
        for i in range(n): 
            vertical[j] += matrix[i][j]
    
    diff = float('inf') 
    
    # 横向切分：从上到下逐行累加，计算差值
    h_cut = 0 
    for i in range(n): 
        h_cut += horizontal[i] 
        diff = min(diff, abs(total - 2 * h_cut))
    
    # 纵向切分：从左到右逐列累加，计算差值
    v_cut = 0 
    for j in range(m): 
        v_cut += vertical[j] 
        diff = min(diff, abs(total - 2 * v_cut))
    
    print(diff) 
```
