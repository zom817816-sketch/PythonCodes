# 哈希表 
## 理论基础 
哈希表是根据关键码的值而直接进行访问的数据结构 
哈希表中关键码就是数组的索引下标，然后通过下标直接访问数组中的元素，如下图所示：![[数组-哈希表.png]]
**一般哈希表都是用来快速判断一个元素是否出现集合里。**
### 哈希函数 
用哈希函数把键（key）通过hashCode转换为数组索引；通过索引直接访问数据，实现 O(1) 平均时间复杂度![[哈希映射.png]]
如果hashcode得出的数值大于哈希表大小，则会进行一次取模的操作；如果键的数量大于哈希表的大小，就一定会产生哈西碰撞 
### 哈希碰撞 
哈希碰撞是指不同的输入数据经过哈希函数计算后，得到相同的输出值 ![[哈希碰撞.png]]
一般哈希碰撞有两种解决方法， 拉链法和线性探测法。
- 链地址法：碰撞的元素形成链表
- 开放地址法：探测其他空位存储冲突数据(如线性探测法)
### 相关数据结构(Python)
 - dict(字典):键必须是可哈希的；Python 3.7以后字典已有序
 - set(集合):底层能使用哈希表存储唯一元素；无序，不重复 
 - frozenset(不可变集合)
 - collections.defaultdict
 - collections.Counter 
 - collections.OrderedDict 
## 有效的字母异位词 
[力扣题目链接](https://leetcode.cn/problems/valid-anagram/)
给定两个字符串 s 和 t ，编写一个函数来判断 t 是否是 s 的字母异位词 
```Python 
def isAnagram(s, t): 
	counter = [0] * 26 
	for i in s: 
		counter[ord(i)-ord('a')] += 1 
	for j in t: 
		counter[ord(j)-ord('a')] -= 1 
	return all(val == 0 for val in counter)
```

```Python 
		return Counter(s) == Counter(t)
```

## 快乐数 
[力扣题目链接](https://leetcode.cn/problems/happy-number/) 
编写一个算法来判断一个数 n 是不是快乐数。
「快乐数」定义为：对于一个正整数，每一次将该数替换为它每个位置上的数字的平方和，然后重复这个过程直到这个数变为 1，也可能是 无限循环 但始终变不到 1。如果 可以变为  1，那么这个数就是快乐数。
如果 n 是快乐数就返回 True ；不是，则返回 False 。
题目中说了会 **无限循环**，那么也就是说**求和的过程中，sum会重复出现，这对解题很重要**。**当我们遇到了要快速判断一个元素是否出现集合里的时候，就要考虑哈希法了**
```Python 
# 时间复杂度O(logn) 空间复杂度O(logn)
def isHappy(n): 
	known_results = set() 
	while n != 1: 
		known_results.add(n) 
		n = sum(int(d)**2 for d in str(n)) 
		if n in known_results: 
			return False 
	return n == 1 
```
```Python 
# 时间复杂度O(logn) 空间复杂度O(1)
# 快慢指针法
def isHappy(n): 
	def get_next(num): 
		return sum(int(d)**2 for d in str(num)) 
	slow, fast = n, get_next(n) 
	
	# 快慢指针：快指针走两步，慢指针走一步
	while fast != 1 and fast != slow: 
		fast = get_next(get_next(fast)) 
		slow = get_next(slow) 
	return fast == 1 
```
## 两数之和 
[力扣题目链接](https://leetcode.cn/problems/two-sum/) 
给定一个整数数组 nums 和一个目标值 target，请你在该数组中找出和为目标值的那 两个 整数，并返回他们的数组下标。
你可以假设每种输入只会对应一个答案。但是，数组中同一个元素不能使用两遍。 

什么时候使用哈希法，当我们需要查询一个元素是否出现过，或者一个元素是否在集合里的时候，就要第一时间想到哈希法。
需要一个集合来存放我们遍历过的元素，然后在遍历数组的时候去询问这个集合，某元素是否遍历过，也就是 是否出现在这个集合。
```Python 
时间复杂度O(n^2) 空间复杂度O(1)
暴力遍历 
def two_sum(nums): 
	for i in range(len(nums)): 
		for j in range(len(nums)): 
			if nums[i] + nums[j] == target: 
				return [i, j] 
	return []
``` 
```Python 
时间复杂度O(n) 空间复杂度O(n)
哈希表 
def two_sum(nums): 
	seen = {} 
	for i, num in enumerate(nums): 
		if target - num in seen: 
			return [seen[target - num], i] 
		seen[num] = i 
	return []
```
## 三数之和 
[力扣题目链接](https://leetcode.cn/problems/3sum/) 
给你一个包含 n 个整数的数组 nums，判断 nums 中是否存在三个元素 a，b，c ，使得 a + b + c = 0 ？请你找出所有满足条件且不重复的三元组。
注意：答案中不可以包含重复的三元组。 
首先将数组排序，然后有一层for循环，i从下标0的地方开始，同时定一个下标left 定义在i+1的位置上，定义下标right 在数组结尾的位置上。
依然还是在数组中找到 abc 使得a + b +c =0，我们这里相当于 a = nums[i]，b = nums[left]，c = nums[right]。
接下来如何移动left 和right呢， 如果nums[i] + nums[left] + nums[right] > 0 就说明 此时三数之和大了，因为数组是排序后了，所以right下标就应该向左移动，这样才能让三数之和小一些。
如果 nums[i] + nums[left] + nums[right] < 0 说明 此时 三数之和小了，left 就向右移动，才能让三数之和大一些，直到left与right相遇为止。 
```Python
# 排序+双指针
# 时间复杂度O(n^2) 空间复杂度O(1)
def three_sum(nums: list[int]) -> list[list[int]]: 
	nums.sort() 
	n = len(nums) 
	res = [] 
	for a in range(n): 
		if nums[a] > 0: 
			break 
		if a > 0 and nums[a] == nums[a-1]: 
			continue 
		b, c = a+1, n-1
		while b < c: 
			total = nums[a] + nums[b] + nums[c] 
			if total > 0: 
				c -= 1 
			elif total < 0: 
				b += 1 
			else: 
				res.append([nums[a], nums[b], nums[c]]) 
				while b < c and nums[b] == nums[b+1]: 
					b += 1 
				while b < c and nums[c] == nums[c-1]: 
					c -= 1 
				b += 1 
				c -= 1 
	return res
```
```Python
# 排序+哈希表
# 时间复杂度O(n^2) 空间复杂度O(n)
def three_sum(nums: list[int]) -> list[list[int]]: 
	nums.sort() 
	n = len(nums) 
	res = [] 
	for a in range(n): 
		# 提前终止
		if nums[a] > 0: 
			break 
		# 跳过重复的元素
		if a > 0 and nums[a] == nums[a-1]: 
			continue 
		seen = {} 
		for b in range(a+1, n): 
			# 跳过重复的元素
			if b > a+1 and nums[b] == nums[b-1]: 
				continue 
			if -nums[a] - nums[b] in seen: 
				res.append([nums[a], nums[b], -nums[a] - nums[b]]) 
				seen.pop(-nums[a] - nums[b]) 
			else: 
				seen[nums[b]] = b 
	return res
```
## 四数之和
[力扣题目链接](https://leetcode.cn/problems/4sum/) 
题意：给定一个包含 n 个整数的数组 nums 和一个目标值 target，判断 nums 中是否存在四个元素 a，b，c 和 d ，使得 a + b + c + d 的值与 target 相等？找出所有满足条件且不重复的四元组。注意：答案中不可以包含重复的四元组。
```Python 
def four_sum(nums: list[int], target: int) -> list[list[int]]: 
	nums.sort() 
	n = len(nums) 
	res = [] 
	for a in range(n): 
		# 提前终止
		if nums[a] * 4 > target: 
			break 
		# 跳过重复的元素
		if a > 0 and nums[a] == nums[a-1]: 
			continue 
		for b in range(a+1, n): 
			# 提前终止
			if nums[a] + nums[b] * 3 > target: 
				break 
			# 跳过重复的元素
			if b > a+1 and nums[b] == nums[b-1]: 
				continue 
			c, d = b+1, n-1
			while c < d: 
				total = nums[a] + nums[b] + nums[c] + nums[d] 
				if total > target: 
					d -= 1 
				elif total < target: 
					c += 1 
				else: 
					res.append([nums[a], nums[b], nums[c], nums[d]]) 
					# 跳过重复的元素
					while c < d and nums[c] == nums[c+1]: 
						c += 1 
					# 跳过重复的元素
					while c < d and nums[d] == nums[d-1]: 
						d -= 1 
					c += 1 
					d -= 1 
	return res
```
